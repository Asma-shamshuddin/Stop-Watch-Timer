# ⏱️ Stopwatch Timer App

A modern and responsive **Stopwatch Timer App** built using **React + Tailwind CSS** with features like:

* ▶️ Start Timer
* ⏹️ Stop Timer
* 🔄 Reset Timer
* 🚩 Lap Tracking
* 🌙 Dark Mode / ☀️ Light Mode Toggle
* 🏆 Fastest Lap Highlight
* 🐢 Slowest Lap Highlight
* 📱 Fully Responsive Design

This project is designed with a clean UI, smooth user experience, and mobile-friendly layout.

## 📂 Features

### Stopwatch Functions

* Start the timer
* Stop the timer
* Reset the timer
* Save lap timings

### Smart Lap System

* Fastest lap shown in green
* Slowest lap shown in red
* Clean lap layout with labels

### UI Features

* Dark mode and light mode toggle
* Beautiful modern card UI
* Background image support
* Premium button design with icons

### Responsive Design

* Works smoothly on:

  * Mobile 📱
  * Tablet 💻
  * Desktop 🖥️

## 📦 Installation

Clone the project:

```bash
git clone https://github.com/Asma-shamshuddin
```

Go to project folder:

```bash
cd your-project-folder
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
